# 💳 ATM Simulation System (Python)

A simple **menu-driven ATM simulation project** built using Python functions and modular file structure.
This project demonstrates clean coding practices, package architecture, and real-world logic implementation.

---

## 🚀 Features

✨ **Display Balance**
✨ **Deposit Money**
✨ **Withdraw Money**
✨ **Transaction Statement (History Tracking)**

---

## 🧠 Concepts Used

* 📦 Package Structure (Modular Programming)
* 🔁 Infinite Loop (`while True`)
* 🔧 Function-Based Design (No Classes)
* 📊 Shared Data Across Modules
* 🧾 Transaction Record Management

---

## 📁 Project Structure

```
ATM_Project/
│
├── main.py
├── services/
│   ├── balance.py
│   ├── deposit.py
│   ├── withdraw.py
│   ├── statement.py
│
├── data/
│   └── account_data.py
│
└── README.md
```

---

## ▶️ How to Run

1. Make sure Python is installed
2. Open terminal in project folder
3. Run the program:

```bash
python main.py
```
